# To-Do List 
Sistema de gerenciamento de tarefas desenvolvido para a disciplina de FPW (Fundamentos de Programação Web). 
A aplicação permite o controle total de atividades cotidianas através de uma interface funcional e intuitiva.

## Funcionalidades
- Cadastro: Adição de novas tarefas à lista.
- Consulta: Visualização clara de todos os itens registrados.
- Edição: Alteração de informações de tarefas existentes.
- Exclusão: Remoção de tarefas concluídas ou indesejadas.
- Filtros: Sistema de busca para localização rápida de itens específicos.
